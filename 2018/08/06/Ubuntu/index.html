使用SVN客服端

  • 安装

    sudo apt-get install subversion

  • checkout,在想要check的目录下打开终端

    svn co {svn repository url}

  • 提交

    svn commit -m path-to-commit，其中path-to-commit可以为空，成功后会提示更新后的版本号

  • 更新

    svn update

  • 日志

    svn log path

安装AppImage软件

  • 添加执行权限

    chmod a+x *.AppImage

  • 执行它

    ./*.AppImage

切换用户

sudo su

查找当前目录下包含”git”的文本

grep -rn “git”

  • *表示当前目录所有文件，也可以是某个文件名

  • -r 是递归查找

  • -n 是显示行号
  • -R 查找所有文件包含子目录
  • -i 忽略大小写
