磁盘

磁盘位置(sata接口):/dev/sd[a-p]

一个磁盘有多个扇区,第一个扇区最重要,记录了两个重要信息

  • 主要启动记录区(MBR):安装开机启动程序
  • 分割表:记录硬盘分割状态

    分区

  • 主分区:最多四个分区
  • 逻辑分区:编号从5开始(sda5)

开机流程

  • BIOS
  • MBR
  • boot loader(包括:提供选单,载入核心档案,转交其他loader)
  • 核心档案

目录和权限

权限分类

  • User
  • Group
  • Others

    文件权限

    -rw-rw-r– 1 yangztel yangztel 23121212 8月 3 11:08 study.pdf

  • 权限
  • 连接数
  • 拥有者
  • 所在组
  • 大小
  • 最后修改时间
  • 文件名(一”.”开头表示隐藏的文件)

    权限:-rw-rw-r–

    第一个字符:文档类型

  • d:目录
  • -:文件
  • i:链接
  • b:可供存储的接口设备
  • c:串行端口

    后面的九个字符每三个字符代表一组共三组,

  • [rwx]:代表读,写,执行
  • 没有权限则用”-“表示
  • 三组按顺序分别表示:拥有者,所在组,其他人

改变文件权限

  • chgrp:改变文件所属群组
  • chown:改变文件拥有者
  • chmod:改变档案权限

    chgrp

  • 查看所有组

    cat /etc/group

  • 对当前文件执行命令

    chgrp {组名} dirname/filename

  • 执行文件夹下面的所有文件

    chgrp {组名} [-R] dirname/filename

    chown

  • 查看所有用户

    cat /etc/passwd

  • 修改所有者

    chown [-R] 账号名称 档案或目彔
    chown [-R] 账号名称:组名 档案或目彔

chmod

  • 权限

    chmod [-R] xyz 档案或目彔
    xyz:分别为拥有者,所在组,其他人在权限加权值

例:chomd -R 770 a.sh
表示:修改a.sh 的权限为:所有者,所在组有所有权限,其他人无权限
7 = r(4) + w(2) + x(1)
